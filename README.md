# Mbarara Ambient PM2.5 (CAMS)

This repository contains code used to generate weekly ambient PM2.5 estimates for Mbarara, Uganda using the ECMWF CAMS Near Real Time dataset in Google Earth Engine.

Data were exported as weekly averages from 2021–2024.

Repository structure:

gee/
  Google Earth Engine scripts for exporting PM2.5 time series

analysis/
  Documentation for combining CSV files and plotting in Excel

Dataset:
ECMWF CAMS NRT PM2.5
https://developers.google.com/earth-engine/datasets/catalog/ECMWF_CAMS_NRT
